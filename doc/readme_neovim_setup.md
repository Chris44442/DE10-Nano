# Neovim Setup

Install Neovim on Ubuntu:

```bash
sudo snap install nvim --classic
```

Set `alias vim='nvim'` in `~/.bashrc`

Create folder and start neovim:

```bash
cd ~/config
mkdir nvim
cd nvim
vim .
```

In Neovim open a new file with `%` called `init.lua`.

Create a folder with `d` called `lua`.
